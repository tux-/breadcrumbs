Breadcrumbs webcomponent

This is a slightly different take on the awesome breadcrumbs by Adam Argyle. This project is closely modeled on the original work, since I really liked that take on breadcrumbs. Modififications include, but are not limited to:

  • Turned into a webcomponent.
  • Changed layout to be based on a list.
  • Keeping the list right scrolled with javascript, and also do this on initial load.
  • Hiding children on navigation is not included.
  • svg graphics changed to use material design icons.
  • I was unable to find any browser that did select autochange as mentioned in the original work, so support for that is not included.

This demo is only meant to show basic of how it works, not actually be interactive on navigation. For that any page loader could be used to update the breadcrumbs content.
